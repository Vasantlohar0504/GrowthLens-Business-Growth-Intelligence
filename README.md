GrowthLens is a **Python-based business analytics platform** designed to analyze sales data and uncover business growth opportunities through data-driven insights and interactive visualizations.

The project processes large sales datasets, performs exploratory data analysis (EDA), and presents insights through an **interactive analytics dashboard built with Streamlit**.

---

# 🚀 Project Overview
Modern businesses rely on data analytics to understand performance and identify growth opportunities. GrowthLens demonstrates how data analysis can be used to transform raw sales data into meaningful insights.

The project includes:
• Data cleaning and preprocessing  
• Exploratory data analysis (EDA)  
• Trend analysis and business insights  
• Interactive data visualization  
• Business intelligence dashboard  

---

# 📊 Dashboard Features

The interactive dashboard provides:

✔ Total Sales KPI  
✔ Total Profit KPI  
✔ Total Orders KPI  
✔ Sales Trend Analysis  
✔ Category Performance  
✔ Customer Segment Insights  
✔ Top Performing Products  
✔ Regional Sales Distribution  
✔ Profit vs Discount Analysis  
✔ Global Sales Map

# ⚙ Installation
Clone the repository

```
git clone [https://github.com/Vasantlohar0504/growthlens-business-analytics.git](https://github.com/Vasantlohar0504/growthlens-business-analytics.git)
```

Navigate to the project folder
```
cd growthlens-business-analytics
```

Install dependencies
```
pip install -r requirements.txt
```
---

# ▶ Run the Project

Run the data processing pipeline
```
python main.py
```
Launch the analytics dashboard
```
streamlit run dashboard/streamlit_dashboard.py
```
